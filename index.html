Flexbox: uma poderosa ferramenta na construção de layouts

Por Marcelo Paschoal, 16 de maio de 2025.

O que é flexbox?

O Flexbox é um modelo de layout em CSS que facilita a criação de interfaces mais flexíveis e adaptáveis. Ele permite organizar elementos de forma unidimensional — em linha ou em coluna — dentro de um flex container, ou seja, um contêiner que utiliza o Flexbox para controlar diretamente a distribuição de seus elementos filhos. Com ele, é possível definir com facilidade o alinhamento, o tamanho, o espaçamento e a ordem dos itens, o que contribui para layouts mais limpos e eficientes em diferentes tamanhos de tela. O Flexbox também veio para simplificar práticas antigas de criação de layout baseadas em propriedades como float e position, oferecendo uma solução mais intuitiva e poderosa. Para utilizá-lo, basta aplicar display: flex ao elemento contêiner.

Conceitos fundamentais do Flexbox

Mas antes de continuarmos, precisamos definir alguns conceitos importantes que nos auxiliarão no decorrer deste artigo. Entender como funcionam o main axis, cross axis, main size, cross size, main start, main end, cross start e cross end é fundamental para aplicar corretamente as principais propriedades do Flexbox. A seguir, apresentamos a explicação de cada um deles.

  • Eixos (Axis)

    O main axis é o eixo principal no qual os elementos filhos do contêiner flex são dispostos, variando de acordo com a direção definida — em linha ou em coluna. Ele não está relacionado diretamente às orientações horizontal ou vertical, mas sim à propriedade flex-direction, que abordaremos mais à frente. O cross axis, por outro lado, é o eixo perpendicular ao main axis, sendo fundamental para compreender o alinhamento dos elementos filhos por meio da propriedade align-items.

  • Tamanho (Size)

    O main size é o tamanho total do contêiner flex em relação ao main axis, enquanto o cross size é o tamanho total do contêiner flex em relação ao cross axis.

  • Início e fim do contêiner (Start and End)

    Tanto o main start quanto o cross start definem o início de um contêiner em relação ao eixo principal ou ao eixo perpendicular. Por outro lado, o main end e o cross end delimitam o fim do contêiner em relação a esses eixos.

Flex direction

A propriedade flex-direction é responsável por definir a direção dos elementos filhos dentro do container flex. Ao alternar seus valores, é possível organizar os elementos em linha, com o primeiro elemento à esquerda utilizando row, ou de forma invertida com row-reverse. Também é possível dispor os elementos em coluna com column, sendo exibidos de cima para baixo, ou de forma invertida com column-reverse.

Justify content

A propriedade justify-content é responsável por alinhar o conteúdo de um container ao longo de seu main axis, definindo como os elementos serão distribuídos nesse eixo. Ao utilizarmos essa propriedade, temos algumas opções bem interessantes. Com o valor flex-start, os itens são agrupados no início do main axis (também chamado de main start). Já com flex-end, temos o comportamento oposto, com os elementos agrupados no final (main end). Quando usamos center, os itens são centralizados ao longo do eixo principal.

Mas essas não são as únicas opções. Também é possível definir espaçamentos entre os elementos do container. Com space-between, todos os itens são separados por espaços iguais, sendo que o primeiro fica encostado no main start e o último no main end. Com space-around, todos os itens recebem o mesmo espaçamento dos dois lados, o que resulta em um espaço duplo entre os elementos internos e um espaço simples nas extremidades. Por fim, com space-evenly, garantimos que os elementos tenham exatamente os mesmos espaços entre si e também entre as bordas do container.

Exemplo de justify-content:

justify-content: flex-start

Align items

Com a propriedade align-items, conseguimos definir como os elementos filhos de um container flex estarão alinhados ao longo do cross axis. Por padrão, um container flex já vem configurado com essa propriedade definida como stretch, fazendo com que todos os elementos filhos ocupem exatamente todo o espaço do cross axis, assumindo o tamanho do maior eixo perpendicular entre eles. No entanto, essa é uma configuração que pode ser ajustada com valores personalizados, conforme a necessidade do design.

Ao utilizarmos outros valores para a propriedade align-items, o comportamento stretch é desativado, e cada elemento passa a ocupar apenas o seu próprio tamanho ao longo do cross axis. O valor center é utilizado para alinhar todos os itens ao centro. Se quisermos que os itens fiquem alinhados ao início ou ao final do eixo transversal, usamos flex-start e flex-end, respectivamente. Mas ainda há outra possibilidade interessante: podemos alinhar os elementos com base no conteúdo interno de cada um por meio do valor baseline.

Exemplo de align-items:

align-items: stretch
